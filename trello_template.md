# Trello - Template de board pour Apartment Hunter

Listes recommandées:
- Backlog (tâches générales)
- To Do (à faire cette itération)
- In Progress (en cours)
- Review / QA
- Done

Exemples de cartes:
- Backlog: "Collecte dataset A / dataset B" — assigner 1 personne
- To Do: "Nettoyage valeurs manquantes" — estimer 4 heures
- In Progress: "Feature engineering - surface, pièces, localisation" — checklist
- Review: "Validation modèle X sur test set"

Utiliser des étiquettes pour prioriser: Urgent / Moyenne / Basse
